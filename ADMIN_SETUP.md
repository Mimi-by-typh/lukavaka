# Настройка админ-панели

## Доступ к админ-панели

Админ-панель доступна по адресу: `https://lukavaka.vercel.app/admin.html`

## Настройка email админа

### Вариант 1: Через переменные окружения Vercel (рекомендуется)

1. Перейдите в настройки проекта на Vercel
2. Откройте раздел **Settings** → **Environment Variables**
3. Добавьте переменную:
   - **Name**: `ADMIN_EMAIL`
   - **Value**: ваш email адрес (например: `your-email@gmail.com`)
4. Нажмите **Save**
5. Перезапустите деплой (Redeploy)

### Вариант 2: Изменить в коде

Откройте файл `api/admin.js` и измените строку:

```javascript
const ADMIN_EMAILS = [
    process.env.ADMIN_EMAIL || 'your-email@gmail.com' // Замените на ваш email
];
```

## Использование админ-панели

1. Откройте `https://lukavaka.vercel.app/admin.html`
2. Введите ваш email адрес (который указан в `ADMIN_EMAIL`)
3. Нажмите "Войти"
4. После успешного входа вы получите доступ к:
   - Статистике (количество комментариев, онлайн пользователей)
   - Управлению комментариями (просмотр, удаление)

## Функции админ-панели

### Статистика
- Общее количество комментариев
- Количество онлайн пользователей

### Управление комментариями
- Просмотр всех комментариев
- Удаление комментариев
- Информация об авторах и датах

## Безопасность

⚠️ **Важно:**
- Email админ должен быть уникальным и известен только вам
- Токен админа хранится в localStorage (действителен 24 часа)
- В продакшене рекомендуется использовать более безопасную систему авторизации
- Для продакшена рекомендуется добавить пароль в дополнение к email

## Добавление нескольких админов

Чтобы добавить несколько админов, измените `api/admin.js`:

```javascript
const ADMIN_EMAILS = [
    'admin1@example.com',
    'admin2@example.com',
    'admin3@example.com'
];
```

Или используйте переменную окружения с разделителем (требует изменения кода).
